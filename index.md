# An application to track tournament outcomes.
User can:
* Create Teams
* Add new Players
* Create Tournaments
* Save data to either SQL or Text file

[Check it out on Azure!](https://www.instagram.com/easyeigo/) 
<H4> App is available Monday - Friday from 0800 - 1200 CST</H4>
